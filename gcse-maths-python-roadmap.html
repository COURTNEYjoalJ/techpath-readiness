Learning roadmap

Build foundations step by step

Start with confidence in maths and problem solving, then connect those foundations to Python and data projects.

GCSE Maths foundations

Arithmetic, algebra, graphs, percentages, ratios and problem solving for stronger technical confidence.

Python basics

Variables, conditions, loops, functions, lists and files using small examples before larger tasks.

Data handling

Tables, CSV files, basic charts and simple analysis so data starts to feel practical.

Data Science basics

Statistics, patterns, predictions and simple machine learning ideas explained at beginner level.

Beginner projects

Calculator, budget tracker, marks tracker, CSV analysis and mini dashboard ideas for practice.

Portfolio direction

Turn small learning projects into portfolio evidence by explaining the problem, method and result.

Study sequence

A simple starter order

1

Strengthen maths

Practise number skills, algebra, graphs and problem-solving questions until the basics feel stable.

2

Learn Python basics

Use simple programs to practise variables, loops, decisions, functions and lists.

3

Handle data

Read small tables or CSV files, clean values, calculate summaries and create basic charts.

4

Build evidence

Choose a small project, explain what it does, and keep notes that can become portfolio evidence.

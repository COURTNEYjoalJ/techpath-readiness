Privacy

Privacy Policy

This policy explains how the current static version of TechPathReady handles browser storage, feedback links and future advertising updates.

Current site behavior

Static website with local browser storage

TechPathReady is currently a static website. It does not use a backend, database, account system, API key, booking system or payment system. The site does not currently collect names, emails or form submissions.

Local quiz scores

Career Path Guide quiz and readiness scores may be saved locally in your browser using localStorage.

Practice quiz

The homepage practice quiz does not save scores and does not affect Career Path Guide readiness reports.

GCSE starter quiz result

The GCSE Maths and Python Basics starter quiz may save the latest completed result locally in your browser using localStorage. This result is not sent to a server because the site has no backend.

No contact, booking or payment form

The current site does not collect messages through built-in forms, process bookings, take payments or store visitor-submitted contact details.

Clearing data

Saved readiness scores and the GCSE starter quiz result remain in the same browser until local site data is cleared by the user.

Feedback and third parties

GitHub Issues, LinkedIn and possible future ads

Feedback is currently handled through GitHub Issues, and guidance messages may be sent through LinkedIn. GitHub Issues feedback is governed by GitHub's own privacy terms, and LinkedIn messages are handled by LinkedIn under LinkedIn's own privacy terms.

TechPathReady itself does not collect messages or form submissions because it does not have a backend or built-in contact form.

Ads are not currently active on TechPathReady. If ads are added in the future, third-party vendors such as Google may use cookies for ad serving. This privacy policy will be updated before ads are enabled.

Questions

Need to suggest a privacy or site improvement?

Use the contact page for the current feedback route. No built-in contact form is active in this version.

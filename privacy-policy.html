Privacy

Privacy Policy

This policy explains how the current static version of TechPath Readiness handles browser storage, feedback links and future advertising updates.

Current site behavior

Static website with local browser storage

TechPath Readiness is currently a static website. It does not use a backend, database, account system or API key. The site does not currently collect names, emails or form submissions.

Local quiz scores

Career Path Guide quiz and readiness scores may be saved locally in your browser using localStorage.

Practice quiz

The homepage practice quiz does not save scores and does not affect Career Path Guide readiness reports.

No contact form

The current site does not collect messages through built-in forms or store visitor-submitted contact details.

Clearing data

Saved readiness scores remain in the same browser until local site data is cleared by the user.

Feedback and third parties

GitHub Issues and possible future ads

Feedback is currently handled through GitHub Issues. If you use that link, GitHub sign-in may be required and GitHub's own terms and privacy policy apply.

Ads are not currently active on TechPath Readiness. If ads are added in the future, third-party vendors such as Google may use cookies for ad serving. This privacy policy will be updated before ads are enabled.

Questions

Need to suggest a privacy or site improvement?

Use the contact page for the current feedback route. No built-in contact form is active in this version.

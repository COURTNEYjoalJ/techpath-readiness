# CodePath Academy - Programming Learning Website

A simple responsive starter website for learning programming languages.

## Included pages/features
- Homepage hero section
- Programming language cards
- Mini lessons for Python, JavaScript, SQL, HTML/CSS, Java and C++
- Beginner learning roadmap
- Beginner project ideas
- Interactive quiz
- Responsive mobile design

## How to open
Open `index.html` in any browser.

## How to publish free
You can upload these files to:
- GitHub Pages
- Netlify
- Vercel

Programming Languages

Use these mini lessons to strengthen the programming skills that appear in your roadmap.

Python

Best for beginners, data science, AI, automation and backend basics.

  • Variables, loops, functions
  • Lists, dictionaries, files
  • Mini projects and data tasks

JavaScript

Best for websites, interactive pages, browser games and full-stack development.

  • DOM, events, functions
  • Arrays and objects
  • Frontend projects

SQL

Best for databases, analytics, business intelligence and data roles.

  • SELECT, WHERE, GROUP BY
  • Joins and aggregations
  • Dashboard-ready queries

HTML & CSS

Best for building website structure, layouts, landing pages and portfolios.

  • Semantic HTML
  • Responsive CSS
  • Forms and components

Java

Best for object-oriented programming, backend systems and enterprise software.

  • Classes and objects
  • Methods and arrays
  • OOP practice

C++

Best for performance, problem solving, systems programming and competitive coding.

  • Syntax and memory basics
  • Functions and arrays
  • Problem-solving patterns

How it works

From career choice to readiness report

Choose a path

Select a country and career role, then review the skills required for that path.

2

Study the weighted skills and topics that matter most for your selected role.

3

Take skill tests

Answer topic-based questions to measure your current understanding.

4

Review readiness

Use your report to identify strengths, weak areas and your next study steps.
